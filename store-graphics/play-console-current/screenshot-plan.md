## 캡처 환경

- 기기: Samsung Galaxy (Android 디바이스 ID `R3CWC0KB53Z`)
- 해상도: 1080x2340 (480 dpi)
- 빌드: `app-debug.apk` (versionName `1.0.0`)
- 캡처 방식: `adb exec-out screencap -p`

`store-graphics/play-console-current/screenshots/` 폴더에 저장되어 있습니다. 권장 업로드 순서:

- 6장 모두 한국어 UI 기준입니다. 영문 리스팅을 활성화한다면 로케일을 영어로 변경하고 동일한 흐름을 다시 캡처해야 합니다.
- 폼팩터 배포 범위가 태블릿까지 확장되면 7인치 / 10인치 태블릿 캡처가 추가로 필요합니다.
- Play Console은 휴대전화 슬롯에 최소 2장, 권장 4~8장을 받습니다. 위 6장이면 앱 프로모션 자격 요건을 충족합니다.
